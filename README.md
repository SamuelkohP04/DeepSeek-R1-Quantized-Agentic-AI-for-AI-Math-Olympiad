# DeepSeek-Tool-Integrated-Reasoning-for-AI-Math-Olympiad

The goal of the AI Mathematical Olympiad 3 (AIMO3) competition is to create open-source algorithms and models that can solve olympiad-level math problems written in LaTeX format. Participation in this competition will help to advance AI models' mathematical reasoning skills and drive frontier knowledge.

Here is the link to the competition: https://www.kaggle.com/competitions/ai-mathematical-olympiad-progress-prize-3
